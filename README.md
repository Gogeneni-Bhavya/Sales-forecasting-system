# Sales Forecasting Using ARIMA

## Project Description

Sales Forecasting is a Machine Learning and Time Series Analysis project developed using Python.  
This project predicts future sales based on historical sales data using the ARIMA model.

The project helps businesses analyze sales trends and forecast future performance for better decision-making.

---

## Features

- Reads sales dataset from CSV file
- Performs time series analysis
- Converts date column into datetime format
- Visualizes sales trends
- Splits data into training and testing sets
- Builds ARIMA forecasting model
- Predicts future sales values
- Calculates forecasting error using RMSE
- Displays actual vs forecast graph
